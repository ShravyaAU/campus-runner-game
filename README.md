# Campus Runner Game

A real-time, interactive, pixelated web game inspired by the classic T-Rex runner. Includes a side-by-side local gameplay screen and a real-time updating live leaderboard using Socket.io!

## 🎮 Play the Game!

**👉 [Play Now on Render](https://campus-runner-game.onrender.com)**

### Features
- Live Top-10 Leaderboard pushing real-time updates via WebSockets.
- Select from multiple animated pixel characters (Duck, T-Rex, Dog, etc.)
- Fully responsive "No WiFi Island Run 2026" neon styling.
- Local high-score tracking.

### Tech Stack
- Frontend: HTML5, Vanilla JS, CSS (Grid & Glassmorphism UI)
- Backend: Node.js, Express, Socket.io
